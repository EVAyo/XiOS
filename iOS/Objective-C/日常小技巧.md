# 修改Xcode默认版本

[官方文档](https://developer.apple.com/library/archive/technotes/tn2339/_index.html)

```shell
# 此处Xcode.app可替换为其他版本，如XcodeBeta.app
$ sudo xcode-select -switch /Applications/Xcode.app
```
