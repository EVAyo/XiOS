# 制作静态库还是动态库呢？

* 动态库：运行时加载，发布的程序需要一同携带着。
* 静态库：将汇编期生成的目标.o文件和引用的库一同链接为可执行文件，有framework和.a的形式。
    * .a：单纯的二进制文件，需要配合.h文件一同使用。
    * .framework：二进制文件 + 资源文件，可以直接使用。
    * .framework = .a + .h + 资源文件
