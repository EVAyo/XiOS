# static VS class

https://juejin.cn/post/6844903792647864328

- static能修饰`class/struct/enum`的计算属性、存储属性、类型方法;class能修饰类的计算属性和类方法
- static修饰的类方法不能继承；class修饰的类方法可以继承
- 在protocol中要使用static
