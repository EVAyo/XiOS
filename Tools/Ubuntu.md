# 制作启动盘

## 推荐：balenaEtcher

* https://www.balena.io/etcher/
* 选择刚刚下载的镜像，再选择刚刚的U盘作为目标，制作启动盘！！

![](media_Ubuntu/002.png)





# 安装Ubuntu

* 设置U盘为第一启动项，F2进入Bios
* 重启，一步步安装即可！！
